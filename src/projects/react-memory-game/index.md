# React Memory Game
---

<br />

Birbirinden farklı 3 seviye ile kart sayıları artarak oyun seviyesi kolay - orta - zor olmak üzere 3 seviyeden oluşmaktadır. Oyunun amacı aynı kartları eşleştirerek oyunu tamamlamaktır.
